# Images

This directory contains images served to the public. Images should be in JPG or SVG format wherever possible.

# Resolution and Filesize

Wherever possible images should use resolutions with the same aspect ratio.

Images should not exceed 600kb and should target 300kb where possible. Thumbnails should re-use original images.

JPG images should be exported at 80% quality.

The following table gives recommendations.

| Image type | Width (px) | Height (px) | Format | Filesize (kB) |
|---|---|---|---|---|
| Landscape | 1246 | 935 | JPG | 600 |
| Portrait | - | - | JPG | 600 |
| Icon | 48px | 48px | SVG | 80 |
